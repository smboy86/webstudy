★변수(variable) : 변하는 값을 기억하는 기억장소 예) 당신의 나이는 20대 이상입니까 ? yes or no 처음의 값이 한 공간에서 계속 변하는 경우 : 1->3->7->2 ★ 변수명 만드는 규칙 : 사용자 정의 명칭 1. 영어 대소문자 활용, 숫자결합 가능 : apple, Apple, apple123 2. 숫자가 앞쪽에 오면 안됨 : 123apple(x) 3. 대소문자 철저히 구분 : a와 A는 서로 다른 변수 4. 띄어쓰기 안됨 : 띄어쓰기 대신 언더바 활용 ex) a b(x), a_b(0) 5. 특수문자 안됨 (하이픈 안됨, 언더바 가능) : a*b(x), a-b(x), a_b(0) 6. 예약어 안됨 : if(x), function(x), for(x), document(x)
