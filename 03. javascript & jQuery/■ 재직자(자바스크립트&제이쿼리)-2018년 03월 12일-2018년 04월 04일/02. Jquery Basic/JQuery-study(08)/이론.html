[기본형식] $(셀렉터).show(속도) ex)$("div").show(1000) : 1초안에 div태그 보이게 하고 싶은 경우 [기본형식] $(셀렉터).show(속도, 콜백함수) 콜백함수 : 애니메이션 효과 처리가 끝난후 이어서 실행되는 명령 즉 처리가 끝나면 이 함수 호출 속도 : slow, normal, fast ---------------------------------------------------------- :animated 뜻? 애니메이션 효과 처리중인 태그 선택가능 하도록 하는 셀렉터 :not(:animated) : 에니메이션 효과 처리중이 아닌 태그 -------------------------------------------------------------- .fadeTo() 함수는 투명도를 조절하여 움직임을 만들어 내는 함수입니다.
