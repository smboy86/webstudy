Team-coForward

coForward는 서로가 소통 [ 通 ( 통할 통 ) ]을 하여 함께 발전해 나가는 [ 進 ( 나아갈 진 ) ] 웹 저작팀입니다.

Logo에 대하여

coForward의 로고 역시 팀의 가치인 [소통]과 [발전]의 의미를 담고 있습니다. 팀명이 영어의 조합으로 이루어졌다면 팀 로고는 한자의 조합으로 구성하였습니다. 팀 가치를 대표하는 한자로 [ 通 통할 통 ]과 [ 進 ( 나아갈 진 ) ]을 선정하고 다음과 같은 의미를 부여하였습니다.

通 ( 통할 통 )
소통이라는 의미의 한자로 coForward 소통뿐 아니라 팀 외부 클라이언트와의 소통 역시 포함합니다. 더 큰 뜻 coForward가 제작하는 웹 자체가 사용자와의 소통의 도구임을 의미합니다.
進 ( 나아갈 진 )
'앞으로 나아가다'라는 의미를 가진 한자로 팀의 지속적인 발전과 coForward가 제작한 웹 사이트를 통하여 운영자와 사용자가 원하는 목적으로 나아갈 수 있다는 의미를 담았습니다.

위와 같은 의미를 가진 두 한자어를 통합하여 공통적인 부분은 한자의 형태를 그대로 유지하고 서로다른 부분은 디자인 및 기능화하여 coForward의 로고가 완성되었습니다.

로고의 형태적 의미

위와 같은 방법으로 디자인된 coForward의 로고는 형태적으로도 coForward가 지향하는 [소통]과 [함께하는 발전]을 의미를 표현하고 강화하고 있습니다.

  • ① : 콘셉트인 두 개 한자의 다른 부분을 단순화하여 QR 코드로 대체하였습니다. 이는 QR 코드가 기술로써 정보를 쉽게 전달하고자 하는 도구라는 것과 사각형의 형태가 웹 사이트 그것과 같습니다. 각기 다른 색상 3개의 큰 사각형은 웹 사이트를 둘러싸고 있는 사용자, 클라이언트 그리고 coForward로 이 세개 요소가 잘 소통되었을 때 웹 사이트가 정보를 바르게 전달할 수 있다는 내용을 형상화 하였으며 QR 코드는 실제 필요한 내용을 담음으로써 실제 사용될 수 있도록 하여 모바일 등 다양한 환경에서 의미를 가질 수 있도록 하였습니다. QR 코드의 내용 역시 고정하지 않고 필요에 따라 변경하여 사용할 수 있습니다.
  • ② : 두 개 한자의 공통 부분은 한자의 부수로, 일반적으로 '책받침'이라 지칭되며 형태적으로도 ①를 받치고 돋보이게 하고 있습니다. 이처럼 coForward는 다양한 환경에서 정보를 전달하는 웹 -(형상화 ①)을 도와주고 받쳐주는 역할을 할 것을 나타내고 있습니다.
  • ①과 ②는 이미지적 전달에 치중하고 있는 상징적 로고로 팀명의 명확한 전달이 어려운 형태이므로 구체적 팀명을 디자인에 병기함으로써 전달 효과를 높였습니다.
