예제 파일

coForward's Color

HTML5, CSS3등 새로운 웹 기술에 빠르게 대응합니다.

차세대 기능 활용

이 페이지는 HTML5의 새로운 기능인 <canvas>와 CSS3의 새로운 기능을 이용하여 구현되었습니다.
구형 웹 브라우저나 새로운 기능을 지원하지 않는 웹 브라우저에서는 정상적으로 작동하지 않을 수도 있습니다.

  • 단색 이미지 스케일 : 느낌에 맞는 색상을 찾아드립니다.
  • 배색 이미지 스케일 : 느낌에 맞는 배색 패턴을 추천합니다.
  • 디자인 QR 코드 : 모바일 기기에서 편리하게 사용되는 QR코드를 생성합니다.

본 내용은 산업자원부의 지원으로 개발된 I.R.I 컬러 시스템과 Google의 chart API를 이용하여 제작되었습니다.
