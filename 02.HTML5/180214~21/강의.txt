1. 포토샵 가이드라인
  1) 파일 확인 C:\smpark\webstudy\02.HTML5\0 HTML5 CSS3\12. belif(반응형)-태블릿 실습&완성\03. PSD\PC_MAIN\pc_main.psd 파일 작업
  2) 기초 환경 셋팅
    - Views > Clear Guide, Clear slice (가이드, 슬라이스 초기화 )
    - Views > Snap 기능 활성화
    - 눈금자 활성화
    - 눈금자 단위 pixel 체크
  3) 큰 레이어 가이드 라인 잡기 (Shift 눌러서 1px씩 이동)
    - header
    - visual
    - contents
    - footer
  4) 가이드를 기준으로 수치 측정

2. html 코드 기본 골격 생성

  2) assets/css

    - 네이버 웹마스터 도구 참조    http://webmastertool.naver.com/guide/basic_markup.naver

4. 각 구역별 이미지, 폰트 정보 캐치 작업
  1) 이미지 슬라이스
  2) 폰트 정보 확인
    - 폰트 정보 (T 타입툴로 글자 선택 > Window > Character 창에서 정보 확인)
    - 폰트 사이즈
    - 폰트 컬러
