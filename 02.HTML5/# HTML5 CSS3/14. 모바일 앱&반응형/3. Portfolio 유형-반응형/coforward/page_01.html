Team-coForward

coForward는 서로가 소통 [ 通 ( 통할 통 ) ]을 하여 함께 발전해 나가는 [ 進 ( 나아갈 진 ) ] 웹 저작팀입니다.

coForward-팀명에 담긴 의미

coForward 구성원들은 토론을 통하여 우리 팀이 웹 저작을 하며 추구해야 하는 가치를 [소통 - 通 ( 통할 통 ) ]과 [함께하는 발전 - 進 ( 나아갈 진 ) ] 두 가지를 선정하였습니다. 두 가지 가치를 교류와 소통을 의미하는 [communication]과 전진 발전을 의미하는 [forward]를 치환하고 합성하여 우리의 팀명 coForward가 만들어졌습니다.

이렇게 만들어진 coForward의 팀명에는 다음과 같은 의미를 담았습니다.

  • coForward의 팀 가치인 [소통 - 通 ( 통할 통 ) ]과 [발전 - 進 ( 나아갈 진 ) ]의 의미를 담았습니다.
  • 영어의 [co~] 접두사는 "함께", "같이" 라는 의미를 가지고 있습니다. 이는 coForward와 함께하는 모든분들이 함께 발전해 나가길 바라는 의미입니다.
  • 발음적으로는 Go! Forward와 유사한 형태로 진취적이고 적극적으로 앞으로 나가자는 의미를 담았습니다.

coForward는 이름에 담긴 의미와 같이 함께 소통하며 발전하는 웹 사이트를 제작하겠습니다.
