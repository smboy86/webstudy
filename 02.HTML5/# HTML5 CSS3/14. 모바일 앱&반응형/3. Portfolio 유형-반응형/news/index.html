이전 : 2010-9 다음 : 2010-11

Start!! coForward

2010년 10월 함께 근무하는 팀원들은 우리가 웹을 만들어가며 추구해야 할 가치가 무엇인지를 함께 의견을 나누고 고민하였습니다.그리고 "함께 발전해 나간다."라는 문장으로 우리가 추구해야 할 바를 정의하였고 이를 바탕으로 team coForward를 시작합니다.

작성일 : 2010년 10월 4일 (Mon)

coForward

International Seminar On Web Accessibility

2010년 10월 6일 team coforward는 정보문화진흥원이 주관하는 웹 접근성 국제 세미나에 참가하였습니다. W3C의 Web Accessibility Initiative (WAI) 의장 Judy Brewer을 비롯한 외국 분들의 웹접근성에 관한 정책과 연구 사례등을 발표하셨습니다. 웹 접근성에 관한 여러가지 지식을 얻을수 있었던 자리였습니다.

작성일 : 2010년 10월 6일 (Wed)
